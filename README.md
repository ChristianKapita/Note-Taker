# Note-Taker

## Description

This Project consist of note taker  application that allows user to save note with a title and plain text. User can select a given note by his name and delete it.

## Installation

Steps to install application:
1. Clone repository.
2. In the directory with app.js run npm install
3. run node Server.js in your CLI
4. Type http://localhost:3000 in you browser.

The app Note Taker is live on <a><img src=https://img.shields.io/badge/Heroku-430098?style=for-the-badge&logo=heroku&logoColor=white/></a>

```
git clone git@github.com:ChristianKapita/SE-Team-Generator.git
npm instal
node  Server.js

```
